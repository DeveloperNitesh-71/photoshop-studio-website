# Photoshop Studio Website

A modern and responsive website designed for a Photoshop studio to showcase services, portfolio work, and client information.

🚧 **Project Status: Work in Progress**

This project is currently under active development. More pages, features, and enhancements will be added soon.

---

## 🎯 Project Goal

The purpose of this project is to create a professional online presence for a creative studio, including:

* Service showcase
* Portfolio gallery
* Client inquiry system
* Modern and engaging UI design

---

## 🚀 Current Features

* 🎨 Homepage design
* 🖼️ Studio introduction section
* 🧱 Structured UI sections
* Clean and modern design

---

## 🔜 Upcoming Features

* Responsive UI
* Portfolio gallery page
* Service details page
* Contact / Inquiry form with validation
* Animations and scroll effects
* Performance optimization
* Deployment-ready version

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
